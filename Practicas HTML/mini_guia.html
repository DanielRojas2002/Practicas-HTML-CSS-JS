Listas

Exiten dos tipos de lista:

  1. Botones
  2. Numericos

1.Botones (< ul > < /ul>)

Tipos de botones :

  • el que esta por deafault type="Disk"
  • type="square"
  • type="circle"
  • Este es un elemento (< li> < /li >)
  • Este es otro elemento (< li> < /li >)

2.Numericos (< ol > < /ol>)

Tipos de Numericos :

  1. el que esta por deafault type="1"
  2. type="I"
  3. type="A"
  4. type="a"
  1. Este es un elemento (< li> < /li >)
  2. Este es otro elemento (< li> < /li >)

Para hacer tablas su sintaxis seria:

  • < table> Aqui inicia la tabla
    • < tr>
      • < th>Encabezado 1< /th>
      • < th>Encabezado 2< /th>
    • < /tr>
    • < tr>
      • < td> Dato del encabezado 1 < /td>
      • < td> Dato del encabezado 2 < /td>
    • < /tr>
    • < tr>
      • < td> Dato del encabezado 1.1 < /td>
      • < td> Dato del encabezado 2.1 < /td>
    • < /tr>
  • < /table>

Atributos

Atributos Description Ejemplo
Align="left,center,right" Alineacion del elemento < table align="center"> < /table>
width="40% or 40px" El ancho ya sea en porcentaje o en pixeles < table width="40%"> < /table>
height="40% or 40px" El alto ya sea en porcentaje o en pixeles < table height="40%"> < /table>
Border="50px" Crea un borde < table border="50%"> < /table>
bgcolor="red" Especifica un color de fondo < table bgcolor="red"> < /table>
